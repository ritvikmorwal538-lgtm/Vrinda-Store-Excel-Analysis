 # Vrinda Store Data Analysis (Excel Project)

## Project Overview
This project focuses on analyzing sales data of Vrinda Store using Microsoft Excel to extract meaningful business insights.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Graphs
- Data Cleaning Techniques

## Key Analysis Performed
- Sales analysis by gender and age group
- Channel-wise order distribution
- Monthly sales trend analysis
- Top performing categories

## Files Included
- Vrinda Store Data Analysis.xlsx

## Outcome
This project helps in understanding customer behavior and sales performance to support business decisions.
